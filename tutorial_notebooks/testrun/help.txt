usage: trotterExample.py [-h] [--pntsamples PNTSAMPLES]
                         [--pntsinglesamples PNTSINGLESAMPLES]
                         [--depths DEPTHS [DEPTHS ...]] [--onlyTomography]
                         [--persamples PERSAMPLES]
                         [--noise_strengths NOISE_STRENGTHS [NOISE_STRENGTHS ...]]
                         [--expectations EXPECTATIONS [EXPECTATIONS ...]]
                         [--shots SHOTS] [--num_qubits NUM_QUBITS]
                         [--circuitfilename CIRCUITFILENAME]
                         [--circuitfunction CIRCUITFUNCTION] [--cross]
                         [--setqubits SETQUBITS [SETQUBITS ...]]
                         [--foldername_extra FOLDERNAME_EXTRA]
                         [--real_backend]

options:
  -h, --help            show this help message and exit
  --pntsamples PNTSAMPLES
                        How many samples in PNT? Default: 16
  --pntsinglesamples PNTSINGLESAMPLES
                        How many single samples in PNT? Default: 100
  --depths DEPTHS [DEPTHS ...]
                        Decide the depths of the pnt-samples. Default:
                        [2,4,8,16]
  --onlyTomography      Only does the tomography and then ends the program
  --persamples PERSAMPLES
                        How many samples in PER? Default: 1000
  --noise_strengths NOISE_STRENGTHS [NOISE_STRENGTHS ...]
                        Decide the noise strengths at which PER will run.
                        Default: [0.5,1,2]
  --expectations EXPECTATIONS [EXPECTATIONS ...]
                        Decide the expectation values which whill be measured
                        at PER. Default: Z on all used qubits
  --shots SHOTS         How many shots per circuit? Default: 1024
  --num_qubits NUM_QUBITS
                        Define how many qubits the simulated backend should
                        have. Layout: Line? Default: How ever many the biggest
                        circuit has
  --circuitfilename CIRCUITFILENAME
                        Set the name of the file, that contains the function
                        making you circuit. Default: circuit.py
  --circuitfunction CIRCUITFUNCTION
                        Set the name of the function, that return your
                        circuits. The function can only take a backend as
                        input and has to return an array of circuits. Default:
                        make_initial_Circuit
  --cross, -c           Simulates Cross Talk Noise
  --setqubits SETQUBITS [SETQUBITS ...]
                        Which qubits on the backend should be used? When set
                        tries to fullfill request, but does not garantie.
                        Default: 0123 and transpile
  --foldername_extra FOLDERNAME_EXTRA
                        Attach something to the end of the foldernamebase
  --real_backend        Toggels the use of a real IBM Backend. You will be
                        prompted an extra input to confirm this. Use QiskitRun
                        timeService.save_account(channel="ibm_quantum",
                        token="<your_token>") first
